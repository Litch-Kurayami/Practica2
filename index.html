< html >

La eriqueta "< html >" es el elmento raíz de un documento HTML y envuelve todo el contenido de la página web. Indica al navegador web que el documento es un documento HTML y que todo el contenido debe ser interpretado como tal. Esta etiqueta debe estar presente alrededor de todo el contenido de la página.

< p >< !DOCTYPE html > < html lang="es" > < head > < meta charset="UTF-8"> < meta name="description" content="Ejemplo de página web"> < title>Mi Página Web < /head> < body> < h1>Bienvenido a mi página web < p>Esta es una página de ejemplo que utiliza la etiqueta <html> como elemento raíz.

< /body> < /html>
